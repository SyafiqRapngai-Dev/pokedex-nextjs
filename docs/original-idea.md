I want to create a pokedex for the first 151 pokemon. The design should take inspiration from the first pokedex design. The pokemon data is to be fetched from pokeapi.co.

The main features of the pokedex are:

- search by pokedex number (eg. 25 for pikachu)
- navigate left and right
- play the pokemon cry

The pokemon data required from pokeapi are:

- name
- type
- image (use official_artwork.front_default)
- pokemon cry (use cries.latest)
